Cockroach

Testing

To test against Cockroach docker test container set the platform to cockroach in src/test/resources/application-test.yaml

Refer to docs / testing if application-test.yaml doesn't exist yet. 

ebean:
  test:
    platform: cockroach #, h2, postgres, mysql, oracle, sqlserver
    ddlMode: dropCreate # none | dropCreate | migrations | create
    dbName: test
