Manual installation (Linux)

```text cd ~/.local/bin curl https://ebean.io/cli-dist/ebean-init-1.9.jar > ebean-init-1.9.jar curl https://ebean.io/cli-dist/ebeaninit > ebeaninit ln -s ebean-init-1.9.jar ebean-init.jar chmod +x ebeaninit ```
